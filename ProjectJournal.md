# Project Journal

## 23/03/2026

Today I began working on the MCTiers Data Science App. I planned the structure of the program and decided to organise the code into multiple files (`main.py`, `api.py`, `data.py`, `ui.py`, and `history.py`) to improve readability and maintainability.

I created the initial files and started implementing API functionality in `api.py`, focusing on sending requests and retrieving data. I also began developing a basic menu system in `main.py` to allow user interaction.

**Reflection:**
Structuring the program into separate modules early made the project easier to manage and reduced complexity when adding new features later.
