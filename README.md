# MCTiers Data Science App

## Overview
The MCTiers Data Science App is a Python program that allows users to explore Minecraft player rankings using the MCTiers API. The application provides a menu-based interface to view gamemodes, overall rankings, and rankings for specific gamemodes.

## Features
- List all available gamemodes  
- View overall player rankings  
- View rankings by gamemode  
- Find individual player rankings
